## 1. 프로젝트 개요

Touring은  
공공데이터 기반 숙박 정보와  
BeautifulSoup을 활용해 크롤링한 관광지 데이터를 기반으로  
사용자가 숙소를 검색 및 예약하고,  
관광지 정보를 확인할 수 있는 웹 플랫폼입니다.

외부 데이터를 직접 수집하고 가공하여  
실제 서비스처럼 구성하는 것을 목표로 진행한 프로젝트입니다.

---

## 2. 개발 기간  
**2025.10 ~ 2025.11**

---

## 3. 기술 스택

### Backend
- Java  
- Spring MVC  
- JDBC  
- Apache Tomcat  
- Maven  

### Frontend
- JSP  
- CSS / JavaScript  

### Database
- MySQL  

### 데이터 / 외부 연동
- 공공데이터 포털 csv 데이터 
- BeautifulSoup 기반 관광지 데이터 크롤링
- Kakao Maps API (지도 및 위치 시각화)
---

## 4. 주요 기능

### ✅ 사용자 기능
- 회원가입 / 로그인  
- 마이페이지 예약 내역 조회  

---

### ✅ 숙소 기능
- 공공데이터 기반 숙소 정보 제공  
- 숙소 목록 및 상세 페이지 제공  
- 지역 및 카테고리 기반 필터링  

---

### ✅ 관광지 기능
- BeautifulSoup을 이용한 관광지 데이터 크롤링
- 크롤링 데이터 DB 저장 및 관리
- 관광지 목록 및 상세 페이지 제공  

---

### ✅ 예약 기능
- 숙소 예약 기능
- 예약 내역 조회
- 현재 날짜 기준  
  지난 예약 / 예정 예약 자동 분류
- 예약별 리뷰 작성 여부 표시

---

### ✅ 지도 기능
- Kakao 지도 API를 활용한 숙소 위치 표시
- 숙소 상세 페이지에서 위치 마커 출력

---

## 5. 담당 역할

🔹 예약 기능
- 예약 데이터 조회 및 관리  
- 예약 종료일 기준 지난 예약 / 예정 예약 자동 분리 로직 구현
- 예약별 리뷰 작성 여부 분기 처리
